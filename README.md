# Tab Tidy - 重複分頁清理助手 🚀

一個基於 Chrome Manifest V3 開發的高效分頁管理工具。點擊圖示即可一鍵清理重複網址的分頁，並提供完整的管理介面供手動操作。


## 🚀 安裝教學 (開發者模式)

目前本插件尚未上架 Chrome Web Store，您可以透過以下步驟手動安裝：

1. 下載此專案至您的電腦並解壓縮。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右上角的 **「開發者模式 (Developer mode)」**。
4. 點擊左上角的 **「載入解壓縮項目 (Load unpacked)」**。
5. 選擇本專案中的 `src` 資料夾即可完成安裝。

## 📖 使用說明

- **左鍵點擊圖示**：執行快速清理。
- **右鍵點擊圖示**：選擇「開啟重複分頁管理員」進入詳細管理介面。


## ✨ 核心功能

- **一鍵清理**：點擊擴充功能圖示，自動刪除所有重複網址的分頁（保留最新開啟的一個）。
- **即時標記 (Badge)**：圖示右下角即時顯示當前重複的分頁數量。
- **視覺化管理員**：右鍵點擊圖示可開啟獨立管理頁面，查看哪些網址重複及其標題。
- **極致效能**：採用 $O(n)$ 時間複雜度的 Hash Map 比對演算法，即使開啟數百個分頁也不卡頓。

## 🛠️ 技術特色

### 高效率演算法
本插件捨棄了傳統的雙重迴圈比對 ($O(n^2)$)，改用 **Map (Hash Table)** 資料結構進行單次遍歷比對：
- **時間複雜度**：$O(n)$
- **空間複雜度**：$O(n)$
