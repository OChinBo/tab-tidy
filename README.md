# Tab Tidy - 重複分頁清理助手

一個基於 Chrome Manifest V3 開發的高效分頁管理工具。點擊圖示即可一鍵清理重複網址的分頁，並提供完整的檢視介面供手動操作。

**版本：** v1.3

---

## 🚀 安裝教學 (開發者模式)

目前本插件尚未上架 Chrome Web Store，您可以透過以下步驟手動安裝：

1. 下載此專案至您的電腦。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右上角的 **「開發者模式 (Developer mode)」**。
4. 點擊左上角的 **「載入解壓縮項目 (Load unpacked)」**。
5. 選擇本專案中的 `src` 資料夾即可完成安裝。

## 📖 使用說明

| 操作 | 效果 |
|------|------|
| **左鍵點擊**圖示 | 一鍵清理所有重複分頁 |
| **右鍵點擊**圖示 → 開啟重複分頁管理員 | 開啟詳細的分頁檢視中心 |

---

## ✨ 核心功能

- **一鍵清理**：點擊擴充功能圖示，自動刪除所有重複網址的分頁（保留最新開啟的一個）。
- **即時 Badge**：圖示右下角即時顯示當前重複的分頁數量，無重複時自動清空。
- **防抖優化**：批次事件（如 Chrome 重啟時大量分頁恢復）只觸發一次計算，避免崩潰。

---

## 🔍 分頁檢視中心 (Inspect)

右鍵選單開啟管理員後，可看到：

- **統計列**：
  - 總分頁數
  - 獨特網域數
  - 可清理的重複項數量

- **網域分佈**：以標籤形式列出各網域的分頁數量（依數量排序）。

- **重複細節**：列出所有重複 URL 的標題與分頁數量。

- **一鍵清理按鈕**：有重複分頁時出現，執行後重新渲染頁面。

## 🛠️ 技術特色

### 高效率演算法

捨棄傳統雙重迴圈比對 (O(n²))，改用 **Map (Hash Table)** 資料結構進行單次遍歷：

- **時間複雜度**：O(n)
- **空間複雜度**：O(n)

### 清理邏輯

1. 依分頁 ID 由小到大排序（ID 越大越新）
2. 遍歷一次，舊的重複分頁加入待刪除清單
3. 批次執行刪除
